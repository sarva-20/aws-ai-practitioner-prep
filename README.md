## How it works

1. Open `inbox/note.md`
2. Write anything in plain English — what you understood, what tripped you up, key concepts
3. `git push`
4. GitHub Actions classifies your note using Gemini and appends it to the right domain file
5. `inbox/note.md` resets automatically

That's it.
